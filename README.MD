# Formula 1 Telemetry Analysis

This project explores Formula 1 telemetry data using the [FastF1 API](https://github.com/theOehrly/Fast-F1) to analyze patterns in **tire degradation**, **gear shifts per track**, and **lap performance**.

## Overview
- Retrieved and analyzed live F1 telemetry datasets via FastF1.
- Built visual dashboards to compare driver performance across laps.
- Engineered data-cleaning pipelines for large-scale telemetry streams.

## Tech Stack
- **Python**, **Pandas**, **Matplotlib**, **Seaborn**
- **FastF1 API** for telemetry data access
- **Jupyter Notebook** for analysis and visualization
